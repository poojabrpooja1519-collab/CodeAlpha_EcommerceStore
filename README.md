# CodeAlpha_EcommerceStore
A simple e-commerce web application built as part of the CodeAlpha Full Stack Development Internship (Task 1).
## Features
- Product listing with details page
- Shopping cart (add, remove, update quantity)
- User registration and login (JWT authentication)
- Order processing
- MongoDB database integration

## Tech Stack
- **Frontend:** React, React Router, Axios
- **Backend:** Node.js, Express.js
- **Database:** MongoDB (Mongoose)
- **Authentication:** JWT, bcrypt

## How to Run
1. Clone the repository
2. In the `backend` folder: run `npm install`, add your own `.env` file with `MONGO_URI`, `PORT`, and `JWT_SECRET`, then run `node server.js`
3. In the `frontend` folder: run `npm install`, then `npm start`
